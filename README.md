# Lista de Cursos Tech Gratuitos

Uma coleção organizada de cursos gratuitos de tecnologia, programação e desenvolvimento para iniciantes e profissionais em 2026.

![Status](https://img.shields.io/badge/Status-Ativo-success) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## Funcionalidades

- **Filtros inteligentes**: Busque por área, tipo de certificado ou palavras-chave
- **Favoritos**: Salve seus cursos preferidos localmente no navegador
- **Responsivo**: Design adaptado para mobile e desktop
- **Links diretos**: Acesso rápido a todos os cursos
- **Interface moderna**: Design colorido e intuitivo

## Categorias Disponíveis

- **Básico**: Lógica, algoritmos, fundamentos
- **Web**: HTML, CSS, JavaScript, React, Angular
- **Back-end**: Python, PHP, C#, .NET, Node.js
- **Dados**: Análise de dados, Power BI, Bancos de Dados
- **Cloud**: AWS, Azure, DevOps
- **UX / Design**: Figma, UI/UX, Design Gráfico
- **IA**: Machine Learning, Deep Learning
- **Games**: Unity, Desenvolvimento de Jogos
- **Segurança**: Cybersecurity, Linux, Segurança da Informação

## Plataformas Incluídas

- **Curso em Vídeo** (Gustavo Guanabara) - Cursos em português com conteúdo prático
- **freeCodeCamp** - Certificações completas gratuitas
- **Rocketseat** - Cursos modernos de desenvolvimento
- **IBM SkillsBuild** - Fundamentos de desenvolvimento web
- **Microsoft Learn** - Cursos oficiais da Microsoft
- **AWS Training** - Cursos gratuitos de cloud computing
- **Coursera** (auditar grátis) - Cursos de universidades renomadas
- **Data Science Academy** - Análise de dados em português
- **Unity Learn** - Desenvolvimento de games
